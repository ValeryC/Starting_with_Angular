Interpolation

Propriete binding

Le binding de propriété Ce binding permet d'injecter une variable dans une propriété HTML. Pour cela, tu peux utiliser le signe [].

Le binding d'événements

Ce binding permet de passer de l'information du HTML vers le TS. Pour cela, tu peux utiliser le signe (). Tu vas ainsi pouvoir écouter les événements de ton utilisateur, comme par exemple le clic de la souris, ou encore la frappe sur le clavier.

Le binding two-way

Ce binding permet de passer de l'information du HTML vers le TS. Pour cela, tu peux utiliser le signe [()]. Tu vas ainsi pouvoir écouter les événements de ton utilisateur et mettre à jour directement l'information dans le composant et le template HTML.
